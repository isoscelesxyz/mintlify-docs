Aerodrome Finance inherits the contract architecture and full security maintenance from Velodrome V2. Velodrome V2 has been audited and runs a bug-bounty program.

For detailed information about security audits and the bug bounty program, see [Security Audits & Bug Bounty](/overview/security/audits).

For security reports, please reach out to us on [Discord](https://discord.com/invite/aerodrome), or to the contacts provided on [our Github page](https://github.com/aerodrome-finance/).

## Emergency Council, Foundations and Public Goods Fund

To support the health of the protocol and ecosystem, there is a Council consisting of members from the Aerodrome team and prominent figures from within the Optimism and Base community.

The Council can be spawned in case of emergency situations or by passing a governance proposal that requires the council's help to be executed.

The Council address is at `0x99249b10593fCa1Ae9DAE6D4819F1A6dae5C013D` and it can:

- set a new emergencyCouncil
- kill a gauge
- revive a gauge
- set a custom name or symbol for a pool
- activate or deactivate (m)veNFTs (superseded by the governance once enabled)

Important multisigs:

- **Aerodrome Foundation and Incentives**: `0xBDE0c70BdC242577c52dFAD53389F82fd149EA5a`
- **Public Goods Fund**: `0x834C0DA026d5F933C2c18Fa9F8Ba7f1f792fDa52`

## Official Website Mirrors

Aerodrome Finance is accessible through multiple frontend mirrors for redundancy and censorship resistance. Always verify you're using an official domain before connecting your wallet.

### Centralized (Web2)

- [aerodrome.finance](https://aerodrome.finance) - Primary domain

### Decentralized (Web3/IPFS)

- [aero.drome.eth](https://aero.drome.eth) - <Tooltip tip="Ethereum Name Service - decentralized domain names on Ethereum. Learn more at ens.domains">ENS</Tooltip> domain (requires ENS-compatible browser or extension)
- [aero.drome.eth.limo](https://aero.drome.eth.limo) - <Tooltip tip="Ethereum Name Service - decentralized domain names on Ethereum. Learn more at ens.domains">ENS</Tooltip> gateway via eth.limo
- [aero.drome.eth.link](https://aero.drome.eth.link) - <Tooltip tip="Ethereum Name Service - decentralized domain names on Ethereum. Learn more at ens.domains">ENS</Tooltip> gateway via eth.link

<Info>
  <Tooltip tip="InterPlanetary File System - distributed file storage protocol">IPFS</Tooltip>
  mirrors provide censorship-resistant access to the Aerodrome interface. If the primary domains are
  unavailable, you can access the protocol through ENS-based mirrors.
</Info>
