## Contributing Guidelines

### Before Starting Work:

1. **Check GitHub issues** to see if someone is already working on it
2. **Comment on the issue** to claim it (or [create a new issue](https://github.com/aerodrome-finance/docs/issues/new))
3. **Review the [Style Guide](/contributing/style-guide)** for documentation standards
4. **Ask questions** in [Discord](https://discord.gg/aerodrome) [#ðŸ™‹-questions](https://discord.com/channels/1139239009772642416/1139239011169341445) channel

### While Working:

- Follow the [Style Guide](/contributing/style-guide) for voice, tone, and structure
- Use canonical sources for data (never guess addresses, parameters, or formulas)
- Include appropriate risk warnings using `<Warning>` components
- Test your documentation by following it yourself

### After Completion:

- Submit a pull request with a clear description
- Link to the GitHub issue in your PR description
- Respond to review feedback promptly
