This reference page is for **developers, data analysts, and protocol teams who need programmatic or visual access to Aerodrome metrics** for analysis, dashboards, or integration purposes. You'll find verified data sources for TVL, volume, pool performance, and voting data.

## Official sources

### Aerodrome UI

The [Aerodrome application](https://aerodrome.finance) provides real-time data for:

- Pool TVL and volume
- Token prices and liquidity depth
- Individual position performance
- Voting metrics
- Emissions and rewards distribution

### On-chain data via Sugar API

For programmatic access to protocol data, use the [Sugar API](/development), Aerodrome's on-chain contract for efficient batch queries of pools, liquidity, and rewards.

## Third-party analytics platforms

### Dune Analytics

Community-maintained Dune queries and API endpoints for programmatic access to Aerodrome data.

| Metric               | Query                                             | Endpoint                                           |
| -------------------- | ------------------------------------------------- | -------------------------------------------------- |
| **Volume**           | [Query](https://dune.com/queries/4200508/7070156) | `dune.x_drome_analytics.result_aero_vol_daily`     |
| **Fees**             | [Query](https://dune.com/queries/4598494/7664499) | `dune.x_drome_analytics.result_aero_fee_daily`     |
| **TVL**              | [Query](https://dune.com/queries/4174769/7027270) | `dune.x_drome_analytics.result_aero_daily_tvl`     |
| **Voting Rewards**   | [Query](https://dune.com/queries/5792194/9389242) | `dune.x_drome_analytics.result_aero_epoch_stats`   |
| **Emissions**        | [Query](https://dune.com/queries/5183422/8532877) | `dune.x_drome_analytics.result_aero_fed_weekly`    |
| **Pool Performance** | [Query](https://dune.com/queries/3002298)         | `dune.x_drome_analytics.result_aero_unified_query` |

### DefiLlama

[DefiLlama](https://defillama.com/protocol/aerodrome-v2) tracks:

- Total Value Locked (TVL) over time
- Trading volume
- Protocol fees and revenue
- Chain-specific breakdowns (Base vs Optimism)

### Additional platforms

For additional analytics platforms and data sources, check the official Aerodrome community channels or explore blockchain analytics aggregators like Dune Analytics and Token Terminal.
