This reference page lists verified data sources for Aerodrome Finance metrics, including TVL, volume, pool performance, and governance data.

## Primary audience

Developers, data analysts, and protocol teams who need programmatic or visual access to Aerodrome metrics for analysis, dashboards, or integration purposes.

## Official sources

### Aerodrome UI

The [Aerodrome application](https://aerodrome.finance) provides real-time data for:

- Pool TVL and volume
- Token prices and liquidity depth
- Individual position performance
- Voting and governance metrics
- Emissions and rewards distribution

### On-chain data via Sugar API

For programmatic access to protocol data, use the [Sugar API](/development/sugar), Aerodrome's on-chain contract for efficient batch queries of pools, liquidity, and rewards.

## Third-party analytics platforms

### DefiLlama

[DefiLlama](https://defillama.com/protocol/aerodrome-v2) tracks:

- Total Value Locked (TVL) over time
- Trading volume
- Protocol fees and revenue
- Chain-specific breakdowns (Base vs Optimism)
